# LMAP

To run start tag.py and LMAP.py

`python3 tag.py`
`python3 LMAP.py`

### tangoAttacker
This attacker takes as input argument the tags ID number in order to perform calculations. Get this ID from the terminal when starting tag.py
To run it use:

`python3 tangoAttacker.py <ID>`

### Requirements:
`BitVector`
